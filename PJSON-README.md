# rz-ng1-seed 

This is boiler plate for ng modules

## Installation

Download node at [nodejs.org](http://nodejs.org) and install it, if you haven't already.

```sh
npm install rz-ng1-seed --save
```


## Tests

```sh
npm install
npm test
```
